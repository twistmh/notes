#### 动态规划

用`dp[i][j]`来表示(i, j)下标范围内的子串是不是回文子串，那么情况可以分成三种：

1. `if(i == j)`，则一定是回文子串，`dp[i][j] = true`
2. `if(j == i + 1)`，则是一个两字符的子串，只要`s[i] == s[j]`， 就是回文子串
3. 其他情况下，如果`dp[i + 1][j - 1] == true并且s[i] == s[j]`，就是回文子串

```c++
class Solution {
public:
    int countSubstrings(string s) {
        int n = s.size();
        vector<vector<bool>> dp(n, vector<bool>(n, false));
        if(n == 1) return true;
        int ans = 0;
		
        //i和j的循环方式很重要，由于dp[i][j]取决于dp[i + 1][j - 1]，所以i要从大的开始，j要从小的开始
        for(int i = n - 1; i >= 0; i--){
            for(int j = i; j < n; j++){
                if(i == j){
                    dp[i][j] = true;
                    ans++;
                } 
                else if(j == i + 1){
                    if(s[i] == s[j]){
                        dp[i][j] = true;
                        ans++;
                    } 
                }
                else{
                    if(s[i] == s[j] && dp[i + 1][j - 1]){
                        dp[i][j] = true;
                        ans++;
                    } 
                }
            }
        }
        return ans;
    
    }
};
```
