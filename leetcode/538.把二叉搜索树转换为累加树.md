#### 递归

- 每个节点的值都等于自己的值和右子树所有值的和，所有肯定是右，中，左的顺序来遍历

```c++
class Solution {
public:
    int sum = 0;
    TreeNode* convertBST(TreeNode* root) {
        if(!root){
            return NULL;
        }

        //虽然没有任何变量来接收返回值，但是由于convereBST改变的是root->val和sum这两个外部值，所以并不需要返回值
        convertBST(root->right);
        //遍历完右子树后，sum已经是右子树的所有值的和，只需要加起来就可以
        sum += root->val;
        root->val = sum;
        convertBST(root->left);

        return root;
    }
};
```
