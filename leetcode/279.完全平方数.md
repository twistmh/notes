#### 动态规划

- dp[i] 为和为i的完全平方数的最小数量
- 对于i，他的前一个状态数量和比他小的完全平方数的数量有关，所以对于每个i，都需要遍历所有比i小的完全平方数

```c++
class Solution {
public:
    int numSquares(int n) {
        vector<int> dp(n + 1, n + 1);
        //初始化
        dp[0] = 0;

        for(int i = 1; i <=n; i++){
            for(int j = 1; j * j <= i; j++){
                dp[i] = min(dp[i], 1 + dp[i - j*j]);
            }
        }

        return dp[n];
    }
};
```
