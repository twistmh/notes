#### 特殊题

- 对于一个从左到右升序，从上到下升序的矩阵，搜索其中有无一个数

1. 很明显，最最土的方法就是逐行扫描，但是这里都给了一个特殊的矩阵了，肯定不可能还是这样做
2. 正确的方法是从右上角元素开始，如果target比元素小，则左移，比元素大则右移

```c++
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();

        int i = 0;
        int j = n - 1;

        bool ans = false;

        while(i < m && j >= 0){
            if(matrix[i][j] == target){
                ans = true;
                break;
            }
            else if(matrix[i][j] > target){
                j--;
            }
            else{
                i++;
            }
        }

        return ans;
    }
};
```
