#### 递归

- 递归构造二叉树，最外层是包含所有结点，下一层是包含左子树的所有结点和右子树的所有结点，最后一层是叶子结点，每一层递归的关键是找到当前递归的子树包含哪些结点

![image-20260325154425700](C:\Users\lee\AppData\Roaming\Typora\typora-user-images\image-20260325154425700.png)

- 对于一棵树来讲，他的根结点是前序遍历的第一个结点
- 左子树和右子树包含的结点范围如图所示
- 为了通过root的值找到root在中序遍历中的位置，需要用map来创造映射

```c++
class Solution {
public:
    //用来存储中序遍历中元素对应的序号,<元素,对应的序号>
    unordered_map<int, int> pos;

    //递归的是结点的范围，对于最外层的递归，当然要包括所有结点，每一层递归都是一个左子树或者一个右子树，把他们当成一棵树来做构造，最终递归到最里层，也就是一个结点构成的树
    TreeNode* build(vector<int>& preorder, vector<int>& inorder, int pre_left, int pre_right, int in_left, int in_right){
        //如果pre_left大于pre_right，就说明要构造的序列里面根本没有值，也就不需要构造了
        if(pre_left > pre_right){
            return NULL;
        }

        //要构造的序列的前序遍历的第一个值肯定是根节点
        int root_val = preorder[pre_left];
        TreeNode* root = new TreeNode(root_val);

        //在中序遍历中，root的左边全部都是左子树的结点，所以左子树的结点数量可以这样计算
        int root_in_index = pos[root->val];
        int left_size = root_in_index - in_left;

        //左子树的结点在前序遍历中是根节点右边部分的左半部分，在中序遍历中是根节点的左边的部分，结点数量为left_size
        root->left = build(preorder, inorder, pre_left + 1, pre_left + left_size, in_left, root_in_index - 1);

        //右子树的结点在前序遍历中根节点右边部分的右半部分，在中序遍历中是根节点的右边的部分
        root->right = build(preorder, inorder, pre_left + left_size + 1, pre_right, root_in_index + 1, in_right);

        return root;
    }
    
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        int n = inorder.size();
        for(int i = 0; i < n; i++){
            pos[inorder[i]] = i;
        }
        //最外层包含所有结点
        return build(preorder, inorder, 0, n - 1, 0, n - 1);
    }
};
```
