#### 前驱结点法

1. 先找到root的左子树的最右孩子
2. 把root的右子树放到这个孩子的右子树
3. 把root的左子树放到root的右子树
4. 清空root的左子树
5. root = root->right
6. 循环直到root等于NULL为止

```c++
class Solution {
public:
    void flatten(TreeNode* root) {
        TreeNode* curr = root;
        TreeNode* predececor = root;
        while (curr) {
            if (curr->left) {
                predececor = curr->left;
                while (predececor->right) {
                    predececor = predececor->right;
                }
                if (curr->right) {
                    predececor->right = curr->right;
                }
                curr->right = curr->left;
                curr->left = NULL;
            }
            curr = curr->right;
        }
    }
};
```
