#### 特殊解法题

- 先令左边界为0，右边界为n - 1；此时选择一个边界往里面缩

1. 如果让更长的那一边往里缩，那么底边一定会缩短的同时，高也一定不会上升（因为如果新的边比短边长，也是取短边），所以面积只可能会下降，没有意义
2. 从短边的角度来讲，他和1, 2, 3, ....., n-2组成的面积一定不会大于n - 1，也就是说短边能提供的最大面积我们已经获得了，他就没有作用了，所以直接抛弃短边，将短边往里面缩

```c++
class Solution {
public:
    int maxArea(vector<int>& height) {
        int n = height.size();
        int left = 0;
        int right = n - 1;

        int ans = 0;

        while(left < right){
            int s = min(height[right], height[left]) * (right - left);
            ans = max(ans, s);
            if(height[left] <= height[right]){
                left++;
            }
            else{
                right--;
            }
        }
        return ans;
    }
};
```
