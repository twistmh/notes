#### 链表

- 这道题的核心并不在链表，而是在对是否还有后续结点的判断上
- sum由l1->val,l2->val和carry三部分组成，只要这三个部分有一个存在，那么就还有新的结点

```c++
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* dummy = new ListNode;
        ListNode* temp = dummy;

        int carry = 0;

        //sum由l1->val,l2->val和carry三部分组成，只要这三个部分有一个存在，那么就还有新的结点
        while(l1 || l2 || carry){
            //先把carry加上，如果carry不存在（为0），这句话就相当于不存在
            int sum = carry;

            //l1如果还有就加上l1
            if(l1){
                sum += l1->val;
                l1 = l1->next;
            }

            //l2如果还有就加上l2
            if(l2){
                sum += l2->val;
                l2 = l2->next;
            }

            //加入新结点并计算新的carry
            int num = sum%10;
            carry = sum/10;
            temp->next = new ListNode(num);
            temp = temp->next;
        }

        return dummy->next;
    }
};
```
