#### 打家劫舍的二叉树版本

- 二叉树版本的打家劫舍需要用两个状态dp来分别记录偷node和不偷node的状态

- steel和noSteel都没有进行初始化，是因为当node为NULL时，val默认就是0，不需要再初始化

```c++
class Solution {
public:
    unordered_map<TreeNode*, int> steel, noSteel;

    void dfs(TreeNode* root){
        if(root == NULL){
            return;
        }

        dfs(root->left);
        dfs(root->right);

        //如果偷了root，则左右子树都不能偷
        steel[root] = root->val + noSteel[root->left] + noSteel[root->right];
        //如果没偷root，则左右子树可偷可不偷，取最大的情况
        noSteel[root] = max(steel[root->left], noSteel[root->left]) + max(steel[root->right], noSteel[root->right]);
    }

    int rob(TreeNode* root) {
        dfs(root);
        return max(steel[root], noSteel[root]);
    }
};
```
