#### include的作用是什么？

- `#include "head.h"`是将头文件的内容**复制粘贴**到本文件中，也就是说，inlcude头文件如果可以正常运行，那么把他的内容直接写在源文件中也可以正常运行

- include对源文件也有效，效果也是**复制粘贴**，但是一般不会这样操作

------

#### 在项目中，应该如何处理文件之间的调用关系?

如果有sub.h，sub.cpp，main.cpp三个文件，那么sub.cpp和main.cpp都需要include  sub.h

------

#### 既然sub.h被include了两次，为什么不会导致重复定义？

- 在正常情况下，在头文件中不应该**定义**函数或变量，而是**声明**函数或变量
- **声明**可以重复，而**定义**只能一次

```c++
int a;   //声明了变量a
void func(int a, int b);  //声明了函数func

int a = 2;   //定义了变量a
//定义了函数func
void func(int a, int b){

}
```

- 类定义比较特殊，就算因为这种原因导致了重复，编译器也会自行处理
