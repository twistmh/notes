初始化本地仓库

```powershell
git init
```

添加文件到暂存区

```bash
git add 文件名
git add .    目录下所有文件
```

提交到本地仓库

```bash
git commit -m "提交说明"
```

查看提交历史

```bash
git log
```

远程仓库

```bash
#关联远程仓库
git remote add origin 仓库地址
#origin可以叫别的名字但是不推荐

#查看目前关联的远程仓库
git remote -v

#将目前commit到本地仓库的内容推送到远程仓库
git push -u origin master
#master是github的分支名字，如果不存在就会自动创建一个
#在第一次推送后，以后再推送只需要git push，origin和master已经被记住了
```

仓库的更新

```bash
#自动识别目录里面的新文件并且添加到暂存区
git add .

git commit -m "提交说明"

git push
```
